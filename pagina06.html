target

<a href = "https://www.google.com.br/?hl=pt-BR" targt ="_blank">site google </a>

ao clicar vai abrir o google em outra aba

Esse atributo indica para o navegador onde o link deverá ser aberto, se na mesma aba, em outra aba,
ou mesmo em uma nova janela do browser.

Quando o atributo target não é utilizado, seu comportamento padrão é abrir o link na aba atual.

Veja alguns exemplos de uso do target nos Códigos 3 e 4.

_self - ja é o valor padrão

_self é o valor padrão, e vai abrir o link na página atual. Quando não informamos o target,
_self é utilizado.

_blank

_blank é utilizado para abrir links em uma nova aba, porém, o usuário pode mudar o comport -
amento do navegador para que seja aberto em uma nova janela. _blanké indicado para uso com links
externos,assim o usuário acessa o link sem sair da sua página.

Ao clicar no link ira abrir uma aba para o youtube
